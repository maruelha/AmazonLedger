Run all tests with: python -m pytest test_parser.py -v  or double-click run_tests.bat

Fixture A — Multi-order paste (5 orders, order-summary view)

Tests parsing of several orders pasted at once, including a multi-package order, a quantity > 1, digital orders, and a non-B ASIN.

# Test name What it checks
1 test_a_order_count Exactly 5 orders are parsed from the pasted block
2 test_a_dates All 5 dates are converted from German to ISO format
e.g. 10. Januar 2024 → 2024-01-10
3 test_a_totals All 5 order totals are extracted correctly with the German decimal comma
e.g. 49,41 (not 49.41)
4 test_a_types Audible order (Abonnement abgerechnet am) → digital
Castle order (D01-… order number) → digital
All other 3 orders → physisch
5 test_a_packages_for_multi_package_order Order 304-5999696-6159543 has exactly 2 packages:
• UJxfPvsSv: STECKEL + tesa weiss
• U1YGxTsPv: Aigostar + tesa schwarz
6 test_a_aigostar_quantity The Aigostar power strip has quantity 2 (bare number on the line before the item)
7 test_a_book_asin Book ASIN 3837164764 is captured correctly — ASINs don't always start with "B"
8 test_a_prime_video_no_asin Castle (Prime Video) is recognised as an item but has asin = None
9 test_a_noise_links_excluded No item name contains noise phrases such as:
Nochmals kaufen, Schreib eine Produktrezension, Bestelldetails anzeigen, Meine Video-Bibliothek
